Privacy Policy

Tab Bookmark Grouper is designed to work locally in your browser. It does not collect, sell, rent, transmit, or share personal data with external servers.

Policy details

What this policy covers

Data Handled Locally

The extension may access the following browser data only to provide its features:

  • Bookmarks: read to list bookmark folders and open selected bookmark URLs.
  • Open tabs and tab groups: read to group tabs by website and export or restore sessions.
  • Session export files: created locally as .btg files containing window, tab, URL, group title, group color, active-tab, and timestamp information.
  • Session import files: read locally when the user selects a .btg file.
  • Settings: theme and language preference stored in chrome.storage.local.

Data Collection and Sharing

The extension does not send bookmark data, tab URLs, browsing activity, session files, settings, analytics, telemetry, identifiers, or profile information to any external service.

Permissions

  • bookmarks: read bookmark folders and URLs selected by the user.
  • tabs: create tabs, inspect current tab URLs for grouping/export, and restore imported sessions.
  • tabGroups: create, name, color, collapse, and ungroup Chrome tab groups.
  • storage: save local language and theme settings.
  • downloads: save a user-requested .btg session export file.

User Control

Session export files are saved only after the user clicks Export Session. Session import files are read only after the user chooses a file and confirms the import summary.

Users can remove extension settings by using the reset button in the extension settings screen or by uninstalling the extension.
